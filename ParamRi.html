About Me:

Hello, my name is Param Ri. I am a Computer Science student at Georgia Institute of Technology. Like many CS students, I originally got into coding in an attempt to learn how video games are made. Because of this, the first programming language I learned was C++. Although I didn't get far in video game making, the skills I learned with C++ turned out to be useful when are started learning Java at Georgia Perimeter College.
