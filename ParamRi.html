About Me:

Hello, my name is Param Ri. I am a Computer Science student at Georgia Institute of Technology. Like many CS students, I originally got into coding in an attempt to learn how video games were made. Because of this, the first programming language I learned was C++. Although I didn't get far in video game making, the skills I learned with C++ turned out to be usefull when are started learning Java at Georgia Perimeter College. The skills I learned in Java helped me develope a broader understanding of Computer Science. Some of the skills I learned include developing Graphic User Interfaces (GUIs), reading and building XML files, reading from input files, and writing to output files. I earned a high GPA at GPC which allowed me to transfer to Georgia Tech. This semester I am taking Discrete Mathematics, Linear Algebra, Datastructures and Algorithms, and Principles of Microeconomics.
