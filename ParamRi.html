About Me:

Hello, my name is Param Ri. I am a Computer Science student at Georgia Institute of Technology. Like many CS students, I originally got into coding in an attempt to learn how video games are made. Because of this, the first programming language I learned was C++. The basic concepts of object-oriented programming that I learned while playing with C++ proved usefull when I started learning Java at Georgia Perimeter College. The skills I learned in Java helped me develop a broader understanding of Computer Science. Some of the skills I learned include developing Graphic User Interfaces (GUIs), reading and building XML files, reading from input files, and writing to output files. I earned a high GPA at GPC which allowed me to transfer to Georgia Tech. I'm also involved with GITMAD (Georgia Tech Mobile Application Development), a student organization that is all about Android Development. We work on everything from the front-end XML files to the back-end database servers.
